Class: Deferred

MagnetJS. Deferred

A Deferred object handles execution of resolve and reject methods, which trigger the success or error callbacks.

new Deferred()

Methods

<static> all(promises)

Asynchronously execute the specified promises. On completion, return an array of success and error arguments in a 'then' function.
